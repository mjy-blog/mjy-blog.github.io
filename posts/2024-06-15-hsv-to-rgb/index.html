HSV to RGB

변환 결과: rgb(0, 0, 0)

공식

HSV는 Hue(색조), Saturation(채도), Value(명도)를 나타내며, RGB는 Red, Green, Blue의 세 가지 기본 색을 조합하여 색을 표현함

변환 과정

  1. 색조(Hue) 해석:
  • HSV의 색조는 색상환에서의 위치를 0에서 1 또는 0에서 360 사이의 값으로 나타냄
  1. 채도와 명도 처리:
  • 채도는 색상의 순수함을 나타내며, 0은 완전한 회색(무채색), 1은 가장 순수한 색상을 의미함
  • 명도는 색상의 밝기를 나타내며, 0은 완전한 검정색, 1은 색상의 가장 밝은 색을 의미함
  1. RGB 값 계산:
  • 먼저 색조를 기준으로 6개의 구간 중 하나를 결정
  • 채도와 명도, 구간에서의 위치에 따라 각 색상 구성 요소의 활성화 정도를 결정 (자세한 부분은 코드 참조)
  1. 결과값 조정:
  • 0에서 255 사이의 범위로 조정

코드

export function hsvToRgb(
  h: number,
  s: number,
  v: number,
): [r: number, g: number, b: number] {
  let r = 0;
  let g = 0;
  let b = 0;

  let type = Math.floor(h * 6);
  let progress = h * 6 - type;
  let p = v * (1 - s);
  let q = v * (1 - progress * s);
  let t = v * (1 - (1 - progress) * s);

  switch (type % 6) {
    case 0:
      [r, g, b] = [v, t, p];
      break;
    case 1:
      [r, g, b] = [q, v, p];
      break;
    case 2:
      [r, g, b] = [p, v, t];
      break;
    case 3:
      [r, g, b] = [p, q, v];
      break;
    case 4:
      [r, g, b] = [t, p, v];
      break;
    case 5:
      [r, g, b] = [v, p, q];
      break;
  }

  return [Math.round(r * 255), Math.round(g * 255), Math.round(b * 255)];
}
